Let talk about difference from Lambda and Proc

Lambda and Proc are all in the Proc object

1.Lambda check the number of argument  ,but Proc don't.

2. They treat return different
	
	return inside Lambda trigger the code right outside of the lambda code

	for exem:

			def lambda_test
  			lam = lambda { return }
  			lam.call
  			puts "Hello world"
			end

			lambda_test                 # calling lambda_test prints 'Hello World'
